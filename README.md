# [joppekoers.nl](Joppekoers.nl)

My very first deployed project with the goal of learning webdev

Tech learned in chronological order 
- git
- Javascript
- NodeJS
- Express
- EJS
- Linux servers
- Docker (compose)
- TypeScript
- ESLint & Prettier
- SvelteKit
- GitHub actions
- Watchtower

## Usage
```shell
cd frontend
npm i
npm run dev
```
Or via docker compose
```
docker compose up --build
```
